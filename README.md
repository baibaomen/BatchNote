# BatchNote

> 批量截图批注工具 - 专门优化与大模型的互动反馈体验

## 项目背景

AI 对话的反馈效率低下 — 当用户阅读 AI 的长篇输出时，往往有多处想法需要反馈，但当前的对话模式强制「线性沟通」：
- 需要逐一截图，然后在一个输入框里用文字描述
- 思维被打断，上下文被割裂
- 表达不精确（"左边那个"、"下面的代码"）

## 核心功能

- 热键呼出 TopMost 窗口
- Ctrl+V 粘贴截图到列表
- 缩略图悬浮显示原图 + 画笔标注
- 拖拽排序、删除、勾选
- 合成大图并复制到剪贴板

## 技术栈

- .NET Framework 4.8 + WinForms
- Windows 10/11
